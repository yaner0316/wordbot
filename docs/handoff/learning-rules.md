# 学习规则与掌握判定

## 当前掌握逻辑

- 释义级别：一次“确定答对”即算掌握
- 猜对不算掌握证据
- 多义词需要每个释义分别掌握
- 答错会清空此前证据
- 掌握后 7 天进入复查
- 复查通过后写成永久掌握
- 永久掌握后不再自动回流

## 规则含义

“掌握”不是按单词整体粗略判断，而是按释义维度判断。一个多义词只有在每个释义都分别达到掌握条件后，才算整体稳定。

“猜对”不能作为证据，避免孩子靠运气通过题目后被误判为掌握。

“答错清空证据”意味着系统更重视最新表现。如果某个释义之前有过掌握证据，但后续答错，需要重新建立掌握证据。

## 复查机制

单个释义掌握后，7 天后进入复查。复查通过后写成永久掌握，之后不再自动回流。

## 待确认

- 7 天周期是否按自然日计算
- 复查失败后状态如何回退
- 永久掌握是否仍允许家长手动修改
- 多用户场景下掌握记录如何区分
