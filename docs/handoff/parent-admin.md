# 家长管理功能说明

## 当前功能

前端已补回一个独立的管理入口，并在首页顶部保留用户与历史入口。

已确认能力：

- 可以走管理接口查询用户和词库
- 可以批量录入单词或词组
- 可以查询某个单词
- 可以手动改状态为 `Pending` 或 `Mastered`
- 录入和状态修改复用现有后端接口

## 设计原则

- 不挤占孩子答题首页
- 家长操作单独收口
- 孩子侧学习流程和家长侧管理流程分离

## 接口对应关系

- 批量录入：`POST /api/admin/addWords`
- 查询单词：`GET /api/word`
- 手动修改状态：`PUT /api/word`
- 用户列表：`GET /api/admin/users`
- 清理测试数据：`POST /api/admin/cleanup`
