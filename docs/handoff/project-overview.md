# 项目总览

## 项目定位

WordBot 是一个英语单词学习与考核系统，当前目标是把产品推进到“可用、可管、可迭代”的状态。

当前核心方向包括：

- 孩子侧：完成单词学习、答题、复习与掌握判定
- 家长侧：管理词库、查询单词、手动调整学习状态
- 后端侧：维护掌握规则、题目生成、提交判定、统计接口
- 性能侧：定位并优化题干改写、释义补齐、飞书表读写等慢点
- 工程侧：保持规则、接口、测试和前端结构的稳定可回归
